# Nutrition Coach Backend

REST API для мобильного приложения Nutrition Coach.

## Запуск
```bash
mvn spring-boot:run
```

## Swagger UI
- http://localhost:8080/swagger-ui/index.html
- http://localhost:8080/v3/api-docs

## Тесты и покрытие
```bash
mvn test jacoco:report
```

Отчет покрытия после сборки:
- `target/site/jacoco/index.html`

## Доступ по умолчанию
- admin@coach.local / admin1234
- demo@coach.local / demo1234
