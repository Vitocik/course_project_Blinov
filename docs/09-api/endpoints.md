# Endpoints

## Основные маршруты
- `POST /api/auth/register`
- `POST /api/auth/login`
- `GET /api/profile/me`
- `PUT /api/profile/me`
- `GET /api/dashboard/summary`
- `GET /api/nutrition/plan`
- `GET /api/workouts/plan`
- `GET/POST/PUT/DELETE /api/meals`
- `GET/POST/PUT/DELETE /api/workout-plans`
- `GET/POST/DELETE /api/progress`
