# OpenAPI

## Назначение
OpenAPI-описание используется для формализации контрактов REST API и упрощает проверку серверной части проекта.

## Основные группы
- auth;
- profile;
- dashboard;
- nutrition;
- workouts;
- progress;
- admin.

## Примечание
Документация API включена в проектную документацию и помогает показать, что серверная часть не только реализована, но и документирована для внешнего использования.
