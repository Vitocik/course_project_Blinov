# Swagger

## Использование
- Swagger UI предназначен для ручной проверки API и демонстрации работы backend.
- Swagger UI: `http://localhost:8080/swagger-ui/index.html`

## Что удобно показывать
- вход и получение токена;
- запросы к защищённым эндпоинтам;
- CRUD-операции;
- ошибки валидации;
- работу административных сценариев.

## Значение для проекта
Swagger позволяет показать, что серверная часть не только реализована, но и документирована как полноценный интерфейс для интеграции и демонстрации.
