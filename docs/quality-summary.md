# Краткое резюме качества проекта

## Что подтверждено
- backend реализован на Spring Boot 3.4.x с Java 21;
- мобильный клиент реализован на React Native / Expo;
- реализованы REST API, JWT, OpenAPI/Swagger и локальное кэширование;
- проект содержит документацию по всем основным этапам;
- имеются тесты и JaCoCo-отчёт по покрытию;
- в репозитории присутствуют все основные артефакты для демонстрации и защиты.

## Почему проект соответствует требованиям
Проект демонстрирует завершённый цикл разработки: от анализа требований до реализации, тестирования и документирования. Он содержит как кодовую часть, так и проектную документацию, что соответствует требованиям курсового проекта и обеспечивает достаточную полноту для защиты.
