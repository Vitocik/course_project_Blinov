# ADR

## ADR-001: Использование PCMEF
Для backend выбрана многослойная архитектура PCMEF, поскольку она помогает разделить ответственность между контроллерами, бизнес-логикой, сущностями и доступом к данным.

## ADR-002: Использование REST
Для обмена между mobile и backend используется REST API в формате JSON, так как это упрощает интеграцию и тестирование.

## ADR-003: Использование PostgreSQL
В качестве основной СУБД используется PostgreSQL как надёжное и хорошо поддерживаемое решение для учебного проекта.
