# Спецификация методов

## Примеры ключевых методов
- `AuthService.login()` — проверка учётных данных;
- `AuthService.register()` — создание нового аккаунта;
- `DashboardService.getSummary()` — формирование сводки;
- `MealService.create/update/delete()` — управление блюдами;
- `WorkoutService.create/update/delete()` — управление тренировками;
- `ProgressService.addEntry()` — добавление записи прогресса.

## Назначение
Документ помогает связать последовательности вызовов с конкретными сервисами и контроллерами.
