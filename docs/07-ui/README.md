# 07. UI

## Визуальный стиль
Приложение использует:
- тёмную тему с мягкими контрастами;
- крупные карточки;
- скругления;
- аккуратные кнопки;
- иконки для быстрой навигации;
- акцент на современный mobile-first интерфейс.

## Навигация
- нижняя tab-bar навигация;
- экран авторизации;
- dashboard;
- profile;
- nutrition;
- workouts;
- progress;
- settings;
- admin.

## UX-решения
- loading states;
- empty states;
- error feedback;
- offline-friendly поведение;
- акцент на понятную структуру контента.
