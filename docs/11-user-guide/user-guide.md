# Руководство пользователя

## Первый запуск
1. Запустите backend через команду из папки backend.
2. Запустите мобильное приложение через Expo или среду разработки.
3. При необходимости укажите адрес backend в настройках приложения.
4. После старта откройте приложение и выполните регистрацию или вход.

## Регистрация и вход
1. Откройте экран регистрации.
2. Укажите email, пароль и имя.
3. После успешной регистрации выполните вход.
4. При повторном запуске приложение использует сохранённый токен.

## Основные экраны
- **Dashboard** — сводка и быстрый доступ к разделам.
- **Profile** — редактирование личных параметров.
- **Meals** — блюда и план питания.
- **Workouts** — тренировки и план тренировок.
- **Progress** — динамика веса и заметки.
- **Admin** — служебный экран для роли `ADMIN`.

## Основные действия
- редактирование профиля;
- просмотр рекомендаций;
- работа с питанием;
- работа с тренировками;
- запись прогресса;
- переключение темы;
- поиск записей по спискам.

## Оффлайн-режим
Если сервер недоступен, приложение использует локальный кэш и остаётся доступным для просмотра ранее сохранённых данных.

## Типовые проблемы
- **Нет соединения** — проверьте адрес backend и сеть.
- **Ошибка авторизации** — повторите вход или очистите локальные данные.
- **Пустой список** — сначала создайте записи в соответствующем разделе.

## Что показывать на защите
1. Регистрацию и вход.
2. Создание/редактирование записи о питании.
3. Создание/редактирование тренировки или плана.
4. Добавление прогресса.
5. Работа в оффлайн-режиме и демонстрацию сохранённых данных.
