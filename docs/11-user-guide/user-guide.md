# Руководство пользователя

## Первый запуск
1. Запустите backend.
2. Запустите mobile-приложение.
3. При необходимости укажите адрес backend.

## Авторизация
- зарегистрируйтесь;
- выполните вход;
- при тестировании можно использовать demo-режим.

## Основные действия
- редактирование профиля;
- просмотр рекомендаций;
- работа с питанием;
- работа с тренировками;
- запись прогресса.

## Offline-режим
Если сервер недоступен, приложение использует локальный кэш для части данных.
