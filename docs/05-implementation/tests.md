# Tests

## Backend tests
В проекте есть тесты для:
- `AuthService`
- `DashboardService`
- `NutritionService`
- `ProgressService`
- `JwtService`
- `ProfileService`
- `WorkoutService`
- application context

## Инструменты
- JUnit 5
- Mockito
- Spring Boot Test
- JaCoCo

## Что стоит показать на защите
- наличие unit tests;
- запуск `mvn test`;
- отчёт покрытия JaCoCo;
- покрытие business logic и security helper classes.
