# Примечания по реализации

## Mobile
- глобальное состояние вынесено в `AppContext`;
- сетевые запросы централизованы в API-слое;
- локальные данные сохраняются в storage.

## Backend
- бизнес-логика вынесена из контроллеров;
- DTO используются для обмена с frontend;
- ошибки обрабатываются централизованно.
