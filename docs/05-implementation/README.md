# 05. Implementation

Раздел описывает практическую реализацию проекта: структура backend, mobile и связующих компонентов.

## Состав
- `code-structure.md`
- `implementation-notes.md`
- `tests.md` — оставлен как часть исходных материалов проекта

## Ключевые решения
- единая точка входа для mobile;
- разделение backend на контроллеры, сервисы, репозитории и DTO;
- использование context-слоя для общего состояния.
