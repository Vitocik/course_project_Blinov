# Контекстная диаграмма

![Контекст приложения](../images/Контекстная_диаграмма.png)

## Описание
Система взаимодействует с пользователем через мобильное приложение.  
Мобильное приложение обращается к backend по REST API.  
Backend хранит данные в PostgreSQL и возвращает JSON-ответы для UI.

## Внешние сущности
- Пользователь
- Администратор
- Mobile App
- Backend API
- PostgreSQL
