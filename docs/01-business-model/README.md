# 01. Business Model

## Цель этапа
На этом этапе определяется бизнес-ценность проекта, круг пользователей и ожидаемый результат.

## Назначение системы
**Nutrition Coach** — это мобильное приложение для:
- подбора питания;
- формирования тренировочных планов;
- отслеживания прогресса;
- персонализации рекомендаций на основе профиля пользователя.

## Целевая аудитория
- студенты и активные пользователи;
- люди, которые контролируют питание и тренировочный процесс;
- администратор/преподаватель, работающий с учебным стендом.

## Бизнес-ценность
Проект закрывает типовую задачу цифрового health&fitness-помощника:
- снижает время на ручной подбор рациона;
- упрощает контроль калорий и макронутриентов;
- даёт пользователю прозрачную структуру прогресса;
- демонстрирует полный цикл клиент–серверного решения в курсовой работе.

## Результат этапа
На выходе формируется контекст системы, список основных акторов и словарь терминов.
