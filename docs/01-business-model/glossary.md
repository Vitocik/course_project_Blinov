# Глоссарий

| Термин | Описание |
|---|---|
| JWT | JSON Web Token для авторизации |
| CRUD | Create, Read, Update, Delete |
| Offline cache | Локальное хранение данных через AsyncStorage |
| Meal | Блюдо или приём пищи в системе питания |
| Workout | Тренировка или тренировочный план |
| Progress | История измерений веса и заметок |
| Dashboard | Главный экран со сводкой показателей |
| Role | Роль пользователя (`USER`, `ADMIN`) |
