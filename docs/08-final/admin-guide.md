# Admin Guide

## Роль ADMIN
Администратор имеет доступ к расширенным возможностям интерфейса.

## Что может видеть admin
- список пользователей;
- ключевые показатели;
- управленческие карточки;
- данные по приложениям и содержимому.

## Рекомендации по защите
На защите удобно показать:
- вход под обычным пользователем;
- вход под admin;
- различие в доступных экранах и данных.
