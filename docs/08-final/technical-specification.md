# Technical Specification

## Назначение
Приложение предназначено для подбора питания и тренировок с учётом параметров пользователя.

## Технические требования
- React Native + Expo;
- TypeScript;
- Axios;
- AsyncStorage;
- React Navigation;
- Java 21;
- Spring Boot 3;
- PostgreSQL;
- JWT;
- Swagger/OpenAPI;
- unit tests.

## Ограничения
- проект должен работать в mobile и backend частях синхронно;
- offline mode должен сохранять локальные данные;
- защищённые эндпоинты доступны только после авторизации.
