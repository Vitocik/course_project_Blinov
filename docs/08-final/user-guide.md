# User Guide

## Первый запуск
1. Запусти backend.
2. Запусти mobile через Expo.
3. Укажи адрес backend, если он отличается от локального.

## Авторизация
- зарегистрируйся;
- войди в аккаунт;
- используй demo-режим при необходимости.

## Основные действия
- редактирование профиля;
- просмотр рекомендаций;
- работа с питанием;
- работа с тренировками;
- запись прогресса;
- переключение темы.

## Оффлайн-режим
Если сервер недоступен, приложение использует локальный кэш и остаётся доступным для просмотра сохранённых данных.
