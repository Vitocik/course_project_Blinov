# 04. Database

## СУБД
В проекте используется PostgreSQL.

## Модель данных
База данных хранит:
- аккаунты пользователей;
- профили;
- блюда;
- тренировочные планы;
- записи прогресса.

## Особенности
- первичные ключи — `BIGSERIAL/IDENTITY`;
- связи `1:1` и `1:N`;
- индексы для поиска по пользователю и идентификаторам;
- поля дат для аудита и истории.
