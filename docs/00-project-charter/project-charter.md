# Паспорт проекта

## Наименование
Мобильное приложение **Nutrition Coach**.

## Предметная область
Приложение помогает пользователю управлять питанием, тренировками и прогрессом в одном клиент-серверном решении.

## Цель проекта
Создать работоспособное мобильное приложение с серверной частью, документацией, тестированием и набором проектных артефактов, достаточных для защиты курсовой работы и демонстрации полного цикла разработки.

## Основные пользователи
- обычный пользователь;
- администратор;
- преподаватель/проверяющий.

## Границы проекта
В проект входят:
- мобильный клиент;
- backend на Spring Boot;
- REST API;
- база данных;
- документация по этапам разработки;
- тесты;
- материалы для демонстрации и защиты.

В проект не входят:
- платёжные сценарии;
- интеграция с внешними фитнес-сервисами;
- push-уведомления;
- публикация в App Store / Google Play.

## Ожидаемый результат
Получить понятный и завершённый учебный проект, который демонстрирует:
- архитектурное разделение слоёв;
- реализацию основных пользовательских сценариев;
- корректную работу с данными;
- наличие документации по требованиям, архитектуре, БД, тестам, UI и развёртыванию.

## Критерии успешности
- приложение запускается;
- пользователь может войти в систему;
- доступны разделы питания, тренировок и прогресса;
- backend отдаёт данные через REST API;
- документация покрывает основные этапы жизненного цикла проекта;
- проект имеет понятную структуру репозитория и набор проектных артефактов для защиты.
