# Паспорт проекта

## Наименование
Мобильное приложение **Nutrition Coach**.

## Цель проекта
Создать приложение для персонализированного подбора питания, тренировок и контроля прогресса пользователя.

## Основные пользователи
- обычный пользователь;
- администратор;
- преподаватель/проверяющий.

## Ожидаемый результат
Рабочее клиент-серверное приложение с документацией, API, базой данных, тестированием и пользовательским руководством.
