# Executive Summary

## Краткое описание проекта
Nutrition Coach — мобильное приложение для управления питанием, тренировками и прогрессом пользователя. Проект объединяет мобильный клиент, серверную часть на Spring Boot и полный набор проектной документации.

## Цель проекта
Создать учебный, но полноценный продукт, который демонстрирует основные этапы разработки программного обеспечения: анализ требований, проектирование, реализацию, тестирование и документирование.

## Ценность для пользователя
- удобный контроль питания и тренировок;
- хранение истории изменений;
- персонализированные рекомендации;
- доступ к данным через мобильное приложение и REST API.

## Ограничения проекта
Проект ориентирован на учебную демонстрацию и не предназначен для коммерческого запуска без дополнительной доработки.
