# ROI / ценность проекта

Проект имеет учебную и демонстрационную ценность.

## Польза
- показывает полный цикл разработки;
- демонстрирует клиент-серверную архитектуру;
- содержит документацию, API, БД и UI;
- удобно защищается как курсовая работа.

## Эффект
- сокращение времени на ручное планирование питания;
- упрощение контроля тренировок и прогресса;
- повышение наглядности работы системы для пользователя.
