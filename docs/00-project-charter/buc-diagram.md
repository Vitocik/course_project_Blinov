# BUC (Business Use Case)

## Основные бизнес-сценарии
- регистрация и авторизация пользователя;
- заполнение профиля;
- получение рекомендаций;
- просмотр и редактирование питания;
- просмотр и редактирование тренировок;
- запись прогресса;
- администрирование системы.

## Результат
Пользователь получает персонализированный сценарий работы с приложением, а система сохраняет данные и возвращает актуальные рекомендации.
