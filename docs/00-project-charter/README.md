# 00. Project Charter

Этот раздел содержит стартовые материалы проекта: назначение системы, границы решения, ключевые участники и базовое описание бизнес-ценности.

## Что лежит в разделе
- `project-charter.md` — паспорт проекта
- `context-diagram.md` — контекстная диаграмма системы
- `buc-diagram.md` — описание бизнес-случаев использования
- `swot.md` — SWOT-анализ
- `roi.md` — оценка ожидаемой ценности
- `glossary.md` — словарь терминов

## Назначение
Nutrition Coach — учебный и прикладной mobile-first проект для подбора питания, тренировок и отслеживания прогресса пользователя.
