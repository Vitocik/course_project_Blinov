# SWOT-анализ

## Strengths
- чёткое разделение mobile и backend;
- понятная предметная область;
- наличие CRUD-сценариев;
- готовые материалы для защиты.

## Weaknesses
- зависимость от backend API;
- ограниченный объём учебного MVP;
- часть данных хранится локально.

## Opportunities
- расширение рекомендаций;
- подключение аналитики и персонализации;
- добавление push-уведомлений.

## Threats
- ошибки синхронизации;
- некорректная авторизация;
- недостаточная полнота пользовательских сценариев.
