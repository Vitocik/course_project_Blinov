# 07. Refactoring

Раздел фиксирует улучшения структуры и качества кода.

## Состав
- `code-smells.md`
- `refactorings.md`
- `static-analysis-report.md`

## Что здесь важно
- найденные запахи кода;
- причина рефакторинга;
- результат изменений;
- принятые паттерны и упрощения;
- выводы статического анализа.
