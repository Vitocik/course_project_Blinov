# 05. Design

## Цель этапа
На этапе дизайна определяются пользовательские сценарии, порядок экранов и логика взаимодействия между ними.

## Что уже реализовано
- tab navigation;
- stack navigation;
- login/register flow;
- dashboard;
- profile;
- nutrition;
- workouts;
- progress;
- settings;
- admin.

## Подход к UX
- тёмная и светлая темы;
- карточная подача данных;
- акцент на readable typography;
- единые отступы и скругления;
- быстрый доступ к ключевым функциям.
