# Требования к окружению

## Backend
- Java 21+;
- Maven;
- PostgreSQL;
- переменные окружения для подключения к БД и security-настроек;
- доступ к порту 8080 для запуска Spring Boot.

## Mobile
- Node.js;
- npm;
- Expo;
- Android Studio / эмулятор или физическое устройство;
- доступ к сети для работы с REST API.

## Дополнительно
- Git;
- современный браузер для Swagger и администрирования;
- локальный или удалённый сервер базы данных.

## Рекомендуемый порядок запуска
1. Запустить backend.
2. Проверить Swagger UI.
3. Запустить мобильный клиент.
4. Выполнить регистрацию и базовые сценарии.
