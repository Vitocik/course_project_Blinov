# JaCoCo

## Назначение
Отчёт JaCoCo используется для демонстрации покрытия тестами.

## Что важно
- покрытие сервисного слоя;
- покрытие helper-классов;
- базовая проверка security и API-логики.

## Текущее состояние покрытия JaCoCo

По последнему отчёту в проекте достигнуты следующие значения:

- **Instruction coverage**: **71%** (1 047 из 3 674 инструкций покрыты);
- **Branch coverage**: **43%** (78 из 138 веток покрыты);
- **Classes**: 54 класса, из них 6 не покрыты;
- **Methods**: 262 метода, из них 109 не покрыты.

### Где покрытие уже сильное
- `dashboard` — 96% instruction, 100% branch;
- `config` — 98% instruction;
- `auth` — 91% instruction, 87% branch;
- `user` — 92% instruction, 75% branch.

### Где ещё есть резерв
- `common` — очень низкое покрытие, 12%;
- `security` — 45% instruction и 7% branch;
- `meal` и `workout` — есть место для добора веток;
- `admin` — покрыт только частично, особенно по branch coverage.

### Что это означает
Показатель **43% branch coverage** уже выше минимальной цели в 40%, а значит покрытие тестами можно использовать как подтверждение базового качества логики и стабильности ключевых сценариев.
